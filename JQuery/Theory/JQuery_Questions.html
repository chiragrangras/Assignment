  1. What is jQuery?

    2. Ans :- JQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

  2. How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation?

    3. Ans :- CSS Using JQuery - The JQuery css() method is used to get the computed value of a CSS property or set one or more CSS properties for the selected elements.
    Add Class and Remove Class - The addClass() or removeClass() methods are used to add the CSS classes when there is a need to add to our webpage when there is some event listener or to create some kind of effect.

    Syntax :-

    • Adding a class:

      $('selector').addClass(class_name);

    • Remove a class:

      $('selector').removeClass(class_name);
    JQuery Animation :- The animate() method is an inbuilt method in jQuery.Which is used to change the state of the element with CSS style.

    Syntax :-
    $(selector).animate({params},speed,callback)
